Projetos realizados

Site institucional UTD

O site foi feito para a UTD, sigla para Univesidade do Trabalho Digital. Uma iniciativa da secretaria de ciência, tecnologia e educação superior do governo estadual do Ceará - SECITECE. O projeto foi desenvolvido com o intuito de divulgar os serviços oferecidos pela instituição de ensino. Desenvolvi esse site na época com a biblioteca Bootstrap em conjunto com a tríade HTML, CSS3 e JS. Clique aqui para acessar o repositório do projeto

Fala Verde luz

Esse projeto foi feito para divulgar as pautas do Instituto Verde Luz, uma ONG que tem várias ações de cunho social e ambientalista pela cidade de Fortaleza. Fala Verde Luz então veio a luz para ser um website gamificado com o intuito de aumentar a visibilidade e participação social nas causas políticas e ambientais do instituto. Nisso, trabalhei como roteirista, animador 2D e programador web em conjunto com colegas da faculdade. Clique aqui para acessar o site de planejamento deste projeto

Sertão Vida ou Morte

Sertão Vida ou Morte é um game cuja premissa apresenta as adversidades enfrentadas por um sertanejo, que tenta sobreviver no sertão cearense, e este procura uma vida melhor em outro lugar. Portanto, o jogador precisa não só gerenciar os elementos relacionados à saúde do personagem, mas também se defender de inimigos, que são alegorias das dificuldades existentes no sertão. Trabalhei como animador 2D e programador web em conjunto com colegas da faculdade. Clique aqui para acessar o site deste projeto
